Quick Start Guide

Get HostingCo running in minutes

1

Clone Repository

git clone https://github.com/AutoBotSolutions/Opensource-Hosting-Developers-Platform.git
2

Install Dependencies

npm run install:all
3

Start Development

npm run dev
4

Access Application

Frontend: http://localhost:3000
Backend: http://localhost:3003
